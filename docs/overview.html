What is partially monotone layer?ÔÉÅ

pmlayer provides a set of partially monotone layers for PyTorch.

Suppose that we want to train a function \(f(x,y) = (x^2 + y^2)/2\) by using a neural network. This function is monotonically increasing with \(x\) and \(y\). We can learn this function by using a standard neural network, but the monotonicity is not guaranteed.

The partially monotone layers are designed to handle monotonicity constraints. Therefore, the output of these layers is guaranteed as monotonically increasing with respect to \(x\) and \(y\).

The HLattice layer is the key layer of pmlayer. In this layer, a \(k \times k\) grid is created, where \(k \geq 2\) is a hyperparameter to specify the granularity of the grid. Each grid point \(f(x',y')\) is trained to learn the value of the function \(f(x',y')\) and they are trained to satisfy the monotonicity constraints.
